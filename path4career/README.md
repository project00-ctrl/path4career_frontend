# Path4Career Platform

Path4Career is a startup guidance platform that helps users explore startup ideas, connect with mentors, and track their progress through guided startup paths.

## Features
- AI Chat Assistant
- Mentor Marketplace
- Mentor Messaging System
- Reputation Score Engine
- Risk Analyzer
- Startup Path Progression

## Tech Stack
Frontend: React + TailwindCSS  
Backend: Node.js + Express  
Database: MongoDB (Planned)

## Installation

Clone the repository

git clone https://github.com/YOURUSERNAME/path4career.git

Install dependencies

npm install

Run the project

npm run dev
